## You App

### Run App With Docker
```sh
    git clone git@github.com:geekbim/YouApp.git
    cd YouApp
    docker-compose up -d --build
```

### Run App Without Docker
```sh
    git clone git@github.com:geekbim/YouApp.git
    cd YouApp
    npm install
    npm run start
```

### Run Test
```sh
    npm run test
```
